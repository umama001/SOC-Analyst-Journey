# Day 2 – Network Observation

## Protocols Observed

- DNS  
DNS is used to translate domain names into IP addresses.

- HTTP/HTTPS  
HTTP and HTTPS are used for web communication between the browser and web servers.

- TCP  
TCP is a transport protocol that ensures reliable communication between devices.

## Packet Example

Source IP: 192.168.100.103  
Destination IP: 192.168.100.1  
Protocol: DNS

Explanation:  
My computer sent a DNS request to find the IP address of the website www.google.com.
